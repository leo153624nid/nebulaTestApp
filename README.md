Тестовое задание на позицию IOS разработчика 


Задача: Необходимо реализовать мобильное приложение по предоставленному макету Figma

Концепция приложения: Приложение состоит из двух основных модулей:
AI Text Chat
Общение с AI в формате текстового чата
Отправка сообщений пользователем
Отображение ответов AI
История переписки
AI Video Generator
Экран создания видео по текстовому запросу
Отправка запроса на генерацию
Отображение состояния генерации
Отображение результата генерации


Требования:
Технические:
UIKit или SwiftUI на выбор
Поддержка актуальных версий iOS (минимум IOS 16)
Чистая архитектура проекта
Код должен быть читаемым и поддерживаемым
Отсутствие критических багов и падений приложения
Нельзя массово использовать ИИ
UI:
Необходимо максимально точно повторить интерфейс согласно макетам Figma:
Отступы
Шрифты
Цвета
Иконки
Анимации
Логика навигации


Анимации:
Плавные переходы между экранами
Корректные состояния загрузки
Отсутствие визуальных лагов
Отсутствие резких изменений интерфейса

Работа с API
Для упрощения тестового задания сервер всегда возвращает одинаковый ответ независимо от содержимого запроса.
Необходимо:
Реализовать сетевой слой.
Выполнять реальные запросы к API.
Корректно обрабатывать состояния:
Loading
Success
Error

Монетизация (Apphud)
Необходимо интегрировать Apphud.
Требования:
Подключить предоставленный Apphud проект.
Реализовать Paywall согласно макету.
Проверять статус подписки пользователя.
Закрыть премиальный контент для пользователей без активной подписки.
Пользователь с активной подпиской должен иметь полный доступ ко всему функционалу приложения.
После покупки доступ должен открываться без перезапуска приложения.

Корректная интеграция Apphud

Производительность приложения
Плавность анимаций
